# MoveOS Software

**Category:** In-House Connected Vehicle OS

---

## Features

- Keyless entry (phone-based)
- Cruise control
- On-screen navigation
- Over-the-air (OTA) software updates
- Vehicle-to-user communication
- Connected vehicle capabilities

---

## Strategic Role

1. **Differentiation** — Software features that ICE scooters can't match
2. **Continuous improvement** — OTA updates mean vehicles get better over time
3. **Customer lock-in** — App + vehicle ecosystem creates switching costs
4. **Data moat** — Riding data, battery performance, usage patterns
5. **Future revenue** — Potential for subscription features, performance upgrades

---

## Integration with Hardware

MoveOS is part of Ola's integrated hardware-software approach:
- [[Technology/Gen 2 Platform]] — Common hardware platform
- MoveOS — Common software layer
- Together: "computer on wheels" vision
